Grid Units:
  • fr: sets the column or row to a fraction of the available space
  • auto: sets the column or row to the width or height of its content automatically
  • %: adjusts the column or row to the percent width of its container

Area Template - Grid Areas.
repeat function lets you define grid-template-columns and grid-template-rows like this grid-template-columns: repeat(3, 1fr); is the same as 1fr 1fr 1fr;
minmax is used to limit the size of items when container changes size, so in the example minmax(90px, 1fr), it will not go smaller than 90px and will grow to 1fr

The repeat comes with an option to auto-fill, this allows you to insert as many rows or columns of your desired size as possible depending on the size of the container.

Then there is auto-fit, that works almost identically to auto-fill. The only difference is that when the container's size exceeds the size of all the items combined, auto-fill keeps inserting empty rows or columns and pushes your items to the side, while auto-fit collapses those empty rows or columns and stretches your items to fit the size of the container.
